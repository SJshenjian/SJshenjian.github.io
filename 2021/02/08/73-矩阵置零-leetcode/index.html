73_矩阵置零_leetcode

2021.02.08 Jian Shen 算法与数据结构  热度 ℃

给定一个 m x n 的矩阵，如果一个元素为 0，则将其所在行和列的所有元素都设为 0。请使用原地算法

输入:
[
[1,1,1],
[1,0,1],
[1,1,1]
]
输出:
[
[1,0,1],
[0,0,0],
[1,0,1]
]
输入:
[
[0,1,2,0],
[3,4,5,2],
[1,3,1,5]
]
输出:
[
[0,0,0,0],
[0,4,5,0],
[0,3,1,0]
]

题解： 1.首行首列是否置0用额外两个变量标志 2.使用首行首列存储其他行列置0标志 3.根据首行首列标志为其他行列置0 4.根据额外变量标志置0首行首列
